# openglDemo

最终目标：实现基于简单光照模型的十二面体真实感显示(flat明暗处理)

## 简介

这是一个计算机图形学的实验项目，主要目的是完成作业和学习OpenGL在Windows上的图形编程策略，学习现代图形学概念和实践。

## 开发环境和调试环境

Visual Studio 2022 (v143)

Windows SDK版本10.0 (最新安装的版本)
